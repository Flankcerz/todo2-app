# Task Management System

Проста система управління задачами на JavaScript, написана з використанням класів (ES6).

## Функціонал
- Створення, оновлення та видалення задач.
- Відстеження дедлайнів (isOverdue).
- Робота з тегами через `Set`.
- Генерація статистики та групування задач за датами.

## Технічний стек
- JavaScript (OOP, Array methods, Date API)
- HTML5 / CSS3

## Як перевірити
1. Склонуйте репозиторій.
2. Відкрийте `index.html`.
3. Перегляньте результати тестів у консолі розробника.
